一平台概述

ConferencePub 是一款面向会议组织方的全流程会刊出版SaaS平台，支持在线编辑、模板管理、团队审稿、印刷下单、电子书生成等一体化服务。

核心价值：一站式服务 | 灵活模板 | 团队协作 | 智能定价

三核心功能模块

3.1 认证系统

  • 邮箱密码登录/注册
  • 微信扫码登录
  • 微信账号绑定
  • JWT token 认证

3.2 项目管理

  • 创建/编辑/删除项目
  • 项目状态: draft → editing → reviewing → finalizing → printing → completed

3.3 文档管理

  • 文档创建/编辑/删除
  • 版本控制 (Revision)
  • 文档状态: draft → review → frozen → printing

四印刷系统

4.1 印刷参数

参数选项
纸张类型铜版纸、哑粉纸、双胶纸、特种纸、PVC等
纸张克重50g - 400g
纸张尺寸A0、A1、A2、A4、A5、16K、32K等
印刷颜色黑白、单色、双彩、全彩
装订方式胶装、骑马订、线圈装、索线胶装、硬壳精装等
后道工艺覆膜、烫金、UV、压纹、冷裱等

4.2 价格计算

  • 默认定价 (无印刷厂定价时使用)
  • 印刷厂自定义定价
  • 平台加价: 默认15% (可配置)
  • 数量折扣: 100本9折 / 500本8折 / 1000本7折
价格流程：用户提交订单 → 计算预估价格(印刷厂单价×1.15) → 印刷厂确认 → 后台确认 → 用户支付
