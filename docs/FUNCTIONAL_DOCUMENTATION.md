## 平台概述

### 系统架构

| 应用 | 端口 | 说明 |
|------|------|------|
| 用户前端 | 3000 | Next.js 用户界面 |
| API服务 | 3001 | NestJS 后端 |
| 印刷厂Portal | 3005 | 印刷厂管理后台 |

### 设计规范

- **主色调**: rgb(91, 107, 230) (蓝紫色)
- **背景色**: #F5F7FB (浅灰白)
- **图标库**: Lucide React
- **布局**: DashboardLayout / PortalLayout

| 功能 | 说明 |
|------|------|

| 位置/尺寸 | 精确数值调整 |
| 撤销/重做 | 历史记录 |
| 导出 | PNG / PDF (A4格式) |
| AI助手 | 生成文案/图片 (演示模式) |
| 网格背景 | 辅助对齐 |

## 文档编辑器

### 技术实现

- **引擎**: Fabric.js 7.2.0
- **类型**: Canvas 编辑器

### 功能详情

#### 工具栏

| 功能 | 快捷键 | 说明 |
|------|--------|------|
| 撤销 | - | 后退一步 |
| 重做 | - | 前进一步 |
| 缩小 | Ctrl+- | 缩小画布 |
| 放大 | Ctrl++ | 放大画布 |
| 滚轮缩放 | Ctrl+滚轮 | 无极缩放 |
| 删除 | - | 删除选中元素 |
| AI助手 | - | 打开AI面板 |
| 保存 | - | 保存到历史 |
| 导出 | - | 下载文件 |

#### 左侧组件面板

| 组件 | 说明 |
|------|------|
| 文本 | 可编辑文字 |
| 矩形 | 方形/圆角矩形 |
| 圆形 | 正圆形 |
| 三角形 | 等腰三角形 |
| 页面尺寸 | A4/A5/Letter/自定义 |

#### 右侧属性面板

| 属性 | 适用 | 说明 |
|------|------|------|
| 字体 | 文本 | 7种字体选择 |
| 位置 X/Y | 所有 | 精确坐标 |
| 尺寸 宽/高 | 图形 | 精确尺寸 |
| 字体大小 | 文本 | 字号调整 |
| 填充颜色 | 所有 | 颜色选择器 |
| 边框颜色 | 图形 | 边框颜色 |
| 边框宽度 | 图形 | 线条粗细 |
| 透明度 | 所有 | 0-100% |

#### AI助手面板

| 功能 | 说明 |
|------|------|
| 生成文案 | 输入提示词生成文字 |
| 生成图片 | 输入描述生成图片 |
| 插入画布 | 一键添加到编辑区 |

#### 导出功能

| 格式 | 用途 | 分辨率 |
|------|------|--------|
| PNG | 屏幕预览 | 2x |
| PDF | 印刷输出 | A4 300DPI |

## 技术栈

### 前端

- **框架**: Next.js )
- **UI14 (App Router库**: Lucide React
- **编辑器**: Fabric.js 7.2.0
- **样式**: CSS-in-JS (inline styles)

### 后端

- **框架**: NestJS
- **数据库**: PostgreSQL (Prisma ORM)
- **认证**: JWT + Passport

### 开发工具

- **包管理**: pnpm
- **构建**: Turborepo
- **版本**: Git
